## 🇧🇷 Português

### 🎯 Objetivo do Projeto
Este projeto nasceu da transição de uma ideia para um produto utilizável. O objetivo central é o estudo e a prática intensa de lógica de programação, manipulação de DOM e arquitetura básica de software. É uma ferramenta funcional para mestres e jogadores de RPG que buscam agilidade.

### 🛠️ Tecnologias
- HTML5 (Estrutura)
- CSS3 (Design e Animações)
- JavaScript Vanilla (Motor de Rolagem)

### 🚀 Como usar
1. Digite expressões como `2d6+3` ou `1d20`.
2. Use os menus rápidos para sistemas como D&D 5e.
3. Clique em **Rolar** e veja o detalhamento dos dados e o total.

## 🇺🇸 English

### 🎯 Project Goal
This project is the result of transforming an idea into a functional product. The main focus is to study and practice programming logic, DOM manipulation, and basic software architecture. It is a practical tool designed for RPG Game Masters and players who need speed during sessions.

### 🛠️ Tech Stack
- HTML5 (Structure)
- CSS3 (Design & Animations)
- Vanilla JavaScript (Dice Engine)

### 🚀 How to use
1. Enter expressions like `2d6+3` or `1d20`.
2. Use quick menus for systems like D&D 5e.
3. Click **Roll** to see the detailed dice results and the total sum.

### 👴 Senior's Corner (Code Review)
**Attention, Senior Devs!** This project is a learning base. If you spot redundancies, security flaws, or refactoring opportunities (especially regarding global variables and string manipulation), please feel free to open an **Issue** or a **Pull Request**. Your mentorship through code is highly appreciated!
